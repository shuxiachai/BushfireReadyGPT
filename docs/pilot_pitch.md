# BushfireReadyGPT Pilot Pitch

## Product Summary

BushfireReadyGPT is an Australia-focused preparedness planning assistant for councils, schools and community resilience teams. It helps users produce structured draft bushfire preparedness reports from a selected geography, target audience, scenario and planning focus.

The current version is designed for pilot evaluation, not operational emergency command. It supports preparedness planning, evidence review and report drafting while keeping live warnings and life-safety decisions with official emergency services.

## Problem

Local organisations often need practical bushfire preparedness material but face several recurring barriers:

- Preparedness guidance is spread across multiple official websites.
- Community vulnerability context can be difficult to translate into a local planning document.
- Draft plans often lack a clear evidence trail, review process or exportable record.
- Schools and councils need repeatable templates for different audiences, not one-off chatbot answers.

## Proposed Solution

BushfireReadyGPT provides a form-first workflow that generates a draft preparedness report and exposes the reasoning behind it.

Core capabilities:

- Structured English preparedness reports.
- All-Australia SA2 / SA3 / SA4 geography selection.
- ABS-derived community context for selected areas.
- Official source register for Queensland, local council and national weather information.
- Multi-agent evidence trail showing profile, data, risk, planning and quality checks.
- Human review checklist and audit JSON export.
- Markdown, PDF and DOCX report outputs.
- Local Ollama operation without an OpenAI API key.

## Target Pilot Users

- Council community resilience officers.
- School emergency coordinators.
- Community workshop facilitators.
- Preparedness education teams.
- Disaster management support staff.

## Pilot Use Cases

### Council Community Preparedness

Generate a draft report for a selected Queensland region, review the evidence trail, confirm official information sources and export the report for internal review.

### School Preparedness

Create a campus-focused bushfire preparedness draft covering evacuation principles, candidate assembly point criteria, staff/student roles, communication and first aid training.

### Community Workshop Material

Generate a practical preparedness handout structure for residents, including household readiness, vulnerable resident considerations, official sources and a short action plan.

## Governance Position

The tool is intentionally positioned as a planning and drafting assistant.

It does not:

- Provide live fire conditions.
- Issue evacuation orders.
- Confirm assembly points as safe.
- Replace Queensland Fire Department, councils, BoM or emergency services.
- Make life-safety decisions.

All generated content should remain in draft status until reviewed and approved by the responsible organisation.

## Pilot Success Measures

- Reviewers can understand the report structure within five minutes.
- Reviewers can identify the evidence and data limitations behind the report.
- The generated report is useful enough to edit, not rewrite from zero.
- The audit JSON and export formats support internal review.
- Reviewers can clearly distinguish preparedness planning from live emergency advice.

## Suggested Pilot Scope

Run a two-week evaluation with one organisation and one scenario:

- One council resilience team, or
- One school emergency coordination team, or
- One community preparedness workshop group.

The pilot should generate 3-5 reports, collect structured feedback and identify the minimum data and governance requirements for a production version.
