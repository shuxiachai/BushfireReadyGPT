# Licence Register

This document summarises the licence assumptions tracked in `data_australia/licence_register.yml`.

It is not legal advice. Before commercial deployment or government procurement, each source needs source-specific review.

## Current Position

| Source | Current project use | Commercial posture |
| --- | --- | --- |
| Australian Bureau of Statistics | SA2 population/context data, ASGS allocation files and LGA reference context | Often CC BY 4.0 when product terms allow; verify exclusions and attribution. |
| Bureau of Meteorology | Official source link and entry-point status only | Do not cache or republish live warning/data content without specific terms or licence agreement. |
| Queensland Government | Official source links and status check entry points | Often CC BY 4.0 unless otherwise noted; verify page-specific exclusions. |
| Cairns Regional Council | Official source link and status check entry point only | Treat as link-only unless written permission or page-specific licence is obtained. |
| Triple Zero | Emergency number reminder and source link | Safe to reference 000, but do not imply endorsement or reproduce branded content without review. |

## Required Before Commercial Use

- Confirm product-level ABS terms for every downloaded dataset.
- Confirm whether BoM data is link-only, API-licensed or restricted.
- Confirm Queensland Government page-specific terms and attribution wording.
- Request or verify Cairns Council permission before reproducing council content.
- Add attribution text to reports and export packages where required.
- Record source URL, download date, local file path and licence review date.

## Safety Boundary

Licence clearance does not make the tool an emergency decision system. Live warnings, fire bans, evacuation orders and life-safety instructions must remain authoritative at official emergency services.
